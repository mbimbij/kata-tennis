# kata-tennis

Le but de ce kata est d'implémenter une méthode permettant de compter le score d'un jeu.

## Pré-requis

* Java 21
* (optionnel) Maven, ou utilisation de Maven Wrapper

## Éxécution des tests
### via maven
`mvn clean test` ou  `./mvnw clean tests`
